# Icon Textures Directory

This directory should contain 16x16 or 32x32 pixel PNG icon textures for:

## Class Icons
These icons appear on the class selection cards:
- `warrior.png` - Sword or armor icon
- `mage.png` - Staff or spell book icon
- `rogue.png` - Dagger or mask icon
- `ranger.png` - Bow or arrow icon
- `tank.png` - Shield icon
- `priest.png` - Cross or healing symbol icon

## Subclass Icons
- `berserker.png` - Axe or rage symbol
- `paladin.png` - Holy sword or cross-shield
- `pyromancer.png` - Fire symbol
- `frostmage.png` - Ice/snowflake symbol
- `assassin.png` - Hidden blade
- `shadowdancer.png` - Shadow/stealth symbol
- `marksman.png` - Crosshair or scope
- `beastmaster.png` - Paw print or animal
- `guardian.png` - Tower shield
- `juggernaut.png` - Heavy armor or hammer
- `cleric.png` - Healing symbol
- `templar.png` - Crusader helmet

## Stat Icons (8x8 pixels recommended)
These icons appear in the stats dropdown:
- `health.png` - Heart symbol
- `mana.png` - Blue droplet or crystal
- `damage.png` - Sword
- `defense.png` - Shield
- `speed.png` - Boot or feather
- `attack_speed.png` - Dual swords
- `cooldown.png` - Clock or timer
- `regen.png` - Plus sign or sparkle

## Skill Icons
Can be added for individual skills based on their IDs.

## Current Status
The GUI system currently uses colored placeholder icons. Adding PNG textures at the paths specified in the code will automatically replace the placeholders with actual icons.

## Icon Style Guidelines
- Use pixel art style (16x16 or 32x32)
- Keep colors vibrant but not oversaturated
- Maintain visual consistency across all icons
- Include slight border or outline for visibility
- Test on black background (skill trees) and colored backgrounds (class cards)
